# PaperMC Auto Updater & Notifier

Linux (Ubuntu等) 環境で稼働する Minecraft PaperMC サーバーを、完全自動で最新バージョン・最新ビルドにアップデートするBashスクリプトです。更新完了時にはDiscordへサイレント通知を送信します。

## ✨ 主な機能
* **完全自動追従**: PaperMC公式API (v2) から常に最新のマイクラバージョンとビルド番号を取得します。1.21 から 1.22 などのメジャーアップデートにも自動で対応します。
* **安全な展開プロセス**: 本番環境 (`minecraft` フォルダ) を汚さないよう、専用の作業用ディレクトリで `libraries` などの必須ファイルを事前に展開します。
* **フェイルセーフ搭載**: ダウンロードしたjarファイルの容量不足（破損）検知や、APIダウン時の安全停止機能を備えています。
* **クリーンインストール**: `systemd` と連携し、サーバーを安全に停止した上で古いエンジンを削除してから新しい `server.jar` を配置するため、ファイルロックによるエラーを防ぎます。
* **Discord通知機能**: 更新が完了すると、Discordの指定チャンネルに `@silent` (通知音なし) で結果を送信します。
